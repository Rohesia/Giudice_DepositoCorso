# Gestionale Ingresso Azienda Interattivo

Questo progetto Python implementa un gestionale per la registrazione degli ingressi e delle uscite dei dipendenti, la gestione dei badge, dei turni e la segnalazione guasti.

## Funzionalit√† principali

- Visualizzare informazioni di tutti i dipendenti
- Registrare manualmente ingressi e uscite

- Segnalare guasti nelle aree aziendali
